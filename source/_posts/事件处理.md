# 写法

ES5

```javascript
var ActionLink = React.createClass({
  handleClick: function(e) {
    e.preventDefault();
    console.log("hehehe");
  },
  render: function() {
    return (
      <a href="#" onClick={this.handleClick}>
        Click me
      </a>
    );
  }
});

ReactDOM.render(<ActionLink />, document.getElementById("root"));
```

纯函数

```javascript
function ActionLink() {
  function handleClick(e) {
    e.preventDefault();
    console.log("The link was clicked.");
  }

  return (
    <a href="#" onClick={handleClick}>
      Click me
    </a>
  );
}
ReactDOM.render(<ActionLink />, document.getElementById("root"));
```

# 开关案例

把 this 绑定到回调函数中

```javascript
class Toggle extends Component {
  constructor() {
    super();
    this.state = {
      isToggleOn: true
    };
    //把this绑定到回调函数
    this.handleClick = this.handleClick.bind(this);
  }
  handleClick() {
    this.setState(prevState => ({
      isToggleOn: !prevState.isToggleOn
    }));
  }
  render() {
    return (
      <button onClick={this.handleClick}>
        {this.state.isToggleOn ? "ON" : "OFF"}
      </button>
    );
  }
}

ReactDOM.render(<Toggle />, document.getElementById("root"));
```

或者 使用属性初始化器语法

```javascript
class Toggle extends Component {
  constructor() {
    super();
    this.state = {
      isToggleOn: true
    };
    // 没有了this绑定
  }
  // 改成用箭头函数
  handleClick = () => {
    this.setState(prevState => ({
      isToggleOn: !prevState.isToggleOn
    }));
  };
  render() {
    return (
      <button onClick={this.handleClick}>
        {this.state.isToggleOn ? "ON" : "OFF"}
      </button>
    );
  }
}

render(<Toggle />, document.getElementById("root"));
```

又或者 回调函数中使用箭头函数

```javascript
class Toggle extends Component {
  constructor() {
    super();
    this.state = {
      isToggleOn: true
    };
  }
  handleClick() {
    this.setState(prevState => ({
      isToggleOn: !prevState.isToggleOn
    }));
  }
  render() {
    return (
      // 绑定箭头函数
      <button onClick={e => this.handleClick(e)}>
        {this.state.isToggleOn ? "ON" : "OFF"}
      </button>
    );
  }
}

render(<Toggle />, document.getElementById("root"));
```

# 向事件处理程序传递参数

两种方式:

```javascript
<button onClick={(e) => this.deleteRow(id, e)}>Delete Row</button>
<button onClick={this.deleteRow.bind(this, id)}>Delete Row</button>
```

例子:

```javascript
class Toggle extends Component {
  constructor() {
    super();
    this.state = {
      name: "hello world"
    };
  }
  preventPop(name, e) {
    e.preventDefault(e);
    alert(name);
  }
  render() {
    return (
      <div>
        <p>hello</p>
        <a
          href="www.baidu.com"
          onClick={this.preventPop.bind(this, this.state.name)}
        >
          Click
        </a>
      </div>
    );
  }
}

render(<Toggle />, document.getElementById("root"));
```
